# MapleInfo 구동
<br>
0. node.js 설치
<br>
1. cd client // 클라이언트 디렉토리 이동
<br>
2. npm install // 노드 모듈 설치
<br>
3. npm start // 클라이언트 구동
<br>
4. cd server // 서버 디렉토리 이동
<br>
5. npm install -g nodemon // nodemon 설치
<br>
6. npm install express // express 설치
<br>
7. nodemon server.js // 서버 구동
